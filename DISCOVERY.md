# Nekt CLI - Discovery Phase

## Project Overview
CLI tool for interacting with the Nekt Platform API (https://docs.nekt.com/platform-api/introduction).

## Discovery Findings

1. **Tech Stack:** Node.js/TypeScript using **oclif**.
2. **Authentication:** API Keys via `x-api-key` header.
3. **Core Functionality:**
   - Listing Sources
   - Listing Transformations
   - Listing Runs
4. **Target Audience:** Developers and AI Agents.
5. **Output Preferences:** Undecided (JSON is likely preferred for AI Agents).
6. **Naming:** `nekt`
